    About Me

    Hey there! My name is Andy and I am a Software Engineer/Developer from Brooklyn, NY. I graduated from Stony Brook University in 2017.
    Interest-wise, I really enjoy coding, headphone browsing, basketball and technology (broad, but anything that can make our lives easier).

    Python has always been one of my favorite programming languages, so when I decided to write my own personal website from scratch, I thought- why not write it in Python?
    It would be a decent alternative to the classic combination of HTML, CSS, and JavaScript.

    So with the aid of the Flask framework, I managed to write the entire back-end logic using Python!

    The tech stack for this website looks something like this:

    • Python (Flask), HTML, CSS (Bootstrap), SQLite
